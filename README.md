# pm-domain-onboarding

외주 PM이 낯선 도메인에 투입될 때, 도메인의 실제 업무 프로세스와 복잡도를 빠르게 파악하고 기획 blind spot을 사전에 잡아내는 스킬.

## Skills (1)

- **domain-deep-dive** — 도메인의 실제 업무 흐름, 숨은 복잡도, 놓치기 쉬운 포인트, 클라이언트에게 물어볼 질문, 기획 위험도를 쉬운 말로 분석. 프로젝트 초기 온보딩이나 도메인 이해가 부족할 때 사용.

## Commands (1)

- `/deep-dive <서비스 설명>` — 도메인 빠르게 파악하기. 웹 리서치를 포함하여 해당 도메인의 실제 업무 프로세스를 조사하고 5개 섹션으로 분석 결과를 제공.

## 사용 예시

```
/deep-dive 빵집에서 식자재 OCR로 찍어서 원가 계산하는 서비스
/deep-dive 중고차 딜러를 위한 차량 상태 검수 자동화 플랫폼
/deep-dive 병원 예약 시스템에서 진료 후 보험 청구까지 자동화
```
